# VoxKitchen v0.2 Roadmap

## 设计原则

- 每个新算子都是独立文件，<150 行
- 新增指标一律写入 `cut.metrics[key]`，pack_jsonl 自动导出（已支持）
- 可选依赖在 `setup()` 中延迟导入，缺了不影响其他算子
- 所有 GPU 算子自动 fallback 到 CPU

---

## 一、音频质量指标（写入 metrics，可用 quality_score_filter 过滤）

| 算子 | metrics key | 说明 | 依赖 | 优先级 |
|------|------------|------|------|--------|
| `dnsmos_score` | `dnsmos_ovrl`, `dnsmos_sig`, `dnsmos_bak`, `dnsmos_p808` | 微软 DNSMOS：P.835 三维度 + P.808 单一 MOS（无参考） | onnxruntime | P0 |
| `utmos_score` | `utmos` | MOS 自然度评分，适合评估 TTS/合成语音质量 | torch | P0 |
| `pitch_stats` | `pitch_mean`, `pitch_std` | PyWorld (dio+stonemask) 基频统计，过滤异常音高 | pyworld | P0 |
| `clipping_detect` | `clipping_ratio` | 检测音频削波（波形截断于 ±1.0），ratio > 0.01 表示有损 | numpy | P0 |
| `speaking_rate` | `speaking_rate`, `silence_ratio` | 语速（字/秒）+ 静音占比，基于 ASR 文本和时长 | -(需先跑 ASR) | P1 |
| `bandwidth_estimate` | `bandwidth_khz` | 检测实际有效带宽（识别伪上采样音频） | torch | P1 |
| `pesq_score` | `pesq_nb`, `pesq_wb` | 有参考语音质量评分（需提供 ref 音频路径） | pesq | P2 |

## 算子设计参考

### pitch_stats 参考实现（来自 get_audio_pitch_by_pyworld.py）

```python
# PyWorld: dio 提取 F0 粗估，stonemask 精修
f0, t = pyworld.dio(audio, sr, frame_period=frame_period)
pitch = pyworld.stonemask(audio, f0, t, sr)

# 过滤无效值
pitch[pitch < 50] = 0
pitch[pitch > 2400] = 0
pitch = pitch[pitch > 0]

# 统计
metrics["pitch_mean"] = np.mean(pitch)
# 归一化标准差（消除音高绝对值的影响）
norm_pitch = (pitch - pitch.min()) / (pitch.max() - pitch.min())
metrics["pitch_std"] = np.std(norm_pitch)
```

### bandwidth_estimate 参考实现（来自 get_audio_pitch_by_pyworld.py）

```python
# STFT → 频率能量 → 找能量骤降点（高频无能量 = 伪上采样）
spec = torch.stft(audio, n_fft=n_fft, hop_length=hop_length, ...)
freq = torch.fft.rfftfreq(n_fft, d=1/sr)
mean_power = (spec.real**2 + spec.imag**2).mean(dim=-1)
# 前i频段 vs 后N-i频段的能量比，找突变点
rate = compute_ratios(mean_power)
cutoff = (torch.log(rate) > 12).argmax()
bandwidth_khz = freq[cutoff].item() * 2 / 1000
```

## 数据流示例（v0.2 完成后的完整 pipeline）

```yaml
stages:
  - { name: to_wav,    op: ffmpeg_convert, args: { target_format: wav } }
  - { name: vad,       op: silero_vad,     args: { threshold: 0.5 } }
  - { name: extract,   op: ffmpeg_convert, args: { target_format: wav } }
  - { name: snr,       op: snr_estimate }
  - { name: dnsmos,    op: dnsmos_score }
  - { name: pitch,     op: pitch_stats }
  - { name: clipping,  op: clipping_detect }
  - { name: asr,       op: whisper_openai_asr, args: { model: turbo } }
  - { name: langid,    op: whisper_langid }
  - { name: gender,    op: gender_classify, args: { method: f0 } }
  - { name: rate,      op: speaking_rate }
  - { name: filter,    op: quality_score_filter, args: { conditions: [
        "duration > 1", "duration < 30",
        "metrics.snr > 10", "metrics.dnsmos_ovrl > 2.5",
        "metrics.clipping_ratio < 0.01" ] } }
  - { name: pack,      op: pack_jsonl }
```
